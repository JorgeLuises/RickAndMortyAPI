# Rick and Morty API Explorer

Este proyecto es una aplicación web desarrollada con Vite, React y TypeScript que consume la API REST de la serie de televisión Rick and Morty. La aplicación permite explorar y visualizar información sobre personajes, episodios y locaciones de la serie de manera interactiva y dinámica.

## Tecnologías y Características

- **Framework:** React 18 con TypeScript
- **Build Tool:** Vite
- **Gestión de Estado:** Context API con useReducer para estados globales
- **Hooks utilizados:** 
  - `useContext` para acceso al estado global
  - `useReducer` para manejo complejo de estado
  - `useState` para estados locales de componentes
  - `useEffect` para efectos secundarios y llamadas a la API
- **Custom Hooks:** Hook personalizado para encapsular la lógica del Reducer
- **Arquitectura:** Estructura de componentes siguiendo las mejores prácticas de React
- **API:** [Rick and Morty API](https://rickandmortyapi.com/)
